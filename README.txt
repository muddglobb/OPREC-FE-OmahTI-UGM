Terimakasih telah mengecek project saya

Pada website ini, selain menggunakan HTML dan CSS, saya juga menggunakan JavaScript untuk responsifitas.
1. Penggunaan function change mode, untuk mengubah tema dari website. Website ini dapat menggunakan light mode dan dark mode. Untuk menggunakannya dapat menekan tombol moon/sun yang ada di sebelah tombol sign in.
2. Penggunaan tombol more dan collapse, untuk menampilkan projects OmahTI lebih lengkap dan kebih sedikit.
3. Penggunaan tombol view more dan view less untuk melihat list contributors.
4. Pada header, saya menambahkan hover pada tombol-tombol navigasi.
5. Website ini sudah responsive, selain mendukung layer desktop dapat mendukung tablet portrait, tablet landscape, hp portrait, dan hp landscape.

Sekian, terimakasih.
